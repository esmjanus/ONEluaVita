NUMBER thread.state ( THREAD  thid)

It gets the thread or process state.

Parameters
thidA thread type object.
Returns
The numeric value 0 if its finalized or not initialized, 1 if its in execution or -1 in case of execution error.
STRING thread.geterror ( THREAD  thid)

It gets the error mesage from the thread or process.

Parameters
thidA thread type object.
Note
It will return a mesage only in case an error ocurred, otherwise will return an empty string.
Returns
