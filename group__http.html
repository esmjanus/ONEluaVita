Detailed Description

Functions related to Hypertext Transfer Protocol. All actions performed by these functions include an automatic integrated manager cookies.

Function Documentation

STRING http.get ( STRING  url)

Download the file indicated a web address to ram.

Parameters
urlThe web address of the file to download.
Returns
If the file was downloaded successfully, it returns the data, additionally the size of them, otherwise it returns nil.
Note
This function can use the onNetGetFile() callback, more info here .
