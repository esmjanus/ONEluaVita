NUMBER cam.zoom ( )

It gets the actual zoom value.

Returns
The actual zoom value between 10 and 32 (1.0x to 3.2x), -1 in case of error.
NIL cam.zoom ( NUMBER  zoom)

Set the actual zoom value on the camera.

Parameters
zoomThe actual zoom value between 10 and 32 (1.0x to 3.2x).
Note
To be able to set the zoom, is important to follow the next indications:
  • The zoom setting will not work when the fps are 120 or lower to 60.
  • __CAM_RES_VGA Zoom will not work.
  • __CAM_RES_QVGA Must have the fps=60, and the zoom value is from 10 to 20 (1.0x-2.0x).
  • __CAM_RES_QQVGA Must have the fps=60, and the zoom value is from 10 to 32 (1.0x-3.2x).
  • __CAM_RES_CIF Must have the fps=60, and the zoom value is from 10 to 16 (1.0x-1.6x).
  • __CAM_RES_QCIF The zoom value is from 10 to 32 (1.0x-3.2x).
  • __CAM_RES_PSP The zoom value is from 10 to 13 (1.0x-1.3x).
  • __CAM_RES_NGP Zoom will not work.
Returns
Nothing.
NUMBER cam.scanqr ( STRING  title,
COLOR  color 
)

Starts the QR code scanner.

Parameters
titleThe title to show on scanning dialogue.
colorThe drawing color for the scanning dialogue, if this line is skipped, black color is default.
Note
This function goes into an internal loop, stops the script until a scan is done or until the user aborts it with circle button.
Returns
In case of success a string with the value of the QR code, otherwise nil.
