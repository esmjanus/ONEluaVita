Management of the motion sensors on the console.

Function Documentation

NIL motion.read ( )

Reads the status of the motion sensors on the console, saving the results in numerical constants. In this way, to know the X axis orientation, by example, we check the constant value motion.orientation.x directly. This is a list of all the available constants:

  • acceleration
    • x (Axis X)
    • y (Axis Y)
    • z (Axis Z)
  • velocity
    • x (Axis X)
    • y (Axis Y)
    • z (Axis Z)
  • orientation
    • x (Axis X)
    • y (Axis Y)
    • z (Axis Z)
Returns
Nothing.
