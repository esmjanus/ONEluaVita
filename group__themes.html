Detailed Description

Livearea Custom Themes management.

Function Documentation

NUMBER themes.install ( STRING  path)

Allows to install a Custom Theme on the system.

Parameters
pathIs the path of the Custom Theme to install.
Returns
In case of success will return the numeric value 1, in case of error will return the numeric value 0.
TABLE themes.list ( )

Retrieves a list of all Themes installed on the PSVita including CustomThemes and Oficial Themes.

Returns
In case of success will return a table with the next fields:
  • id: Is the complete path to where the Theme is installed on the memory (STRING).
  • package: Name of the PNG file of the theme that consists in a thumbnail of the theme (STRING).
  • home: Name of the PNG file of the theme that consists in a preview of the theme. It will appear in your theme info page (STRING).
  • start: Name of the PNG file of the theme that consists in a preview of the theme's lockscreen. It will appear in your theme info page (STRING).
TABLE themes.info ( STRING  path)

Gets the metadata from the XML file of a theme.

Parameters
pathPath to the XML file.
Returns
If success, returns a table with these fields:
  • title: Theme title. "m_title" (STRING).
  • author: Theme author. "m_provider" (STRING).
  • package: Name of the PNG image used as thumbnail for the theme. "m_packageImageFilePath" (STRING)
  • home: Name of the PNG image used as preview show in the theme info window. "m_homePreviewFilePath" (STRING).
  • start: Name of the PNG image used as lockscreen show in the theme info window. "m_startPreviewFilePath" (STRING).
Warning
The previous list describes the most common data, nevertheless the returned table may not have contain all listed fields, this is why is very important to make sure any field exists before trying to use it.
STRING themes.delete ( STRING  id)

Allows to delete a Custom Theme from the system.

Parameters
idIs the complete path to where the Custom Theme is installed, (can be easily retrieved from the table returned by themes.list()).
Returns
In case of error will return a STRING with details about the error type.
Note
This function allows to delete a Custom Theme, it gets deleted from the file app.db, this way the Theme resources stay intact in the path where is installed.
