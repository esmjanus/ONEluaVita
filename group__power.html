NIL power.tick ( NUMBER  mode)

It allows you to reset the timers of the console (suspension, screen, etc).

Parameters
modeAny of the following ways:
  • __POWER_TICK_ALL (0), reset all timers.
  • __POWER_TICK_SUSPEND (1), restarts the timer suspension.
  • __POWER_TICK_OLED_OFF (4), restart the timer off screen.
  • __POWER_TICK_OLED_DIMMING (6), restart timer dimming the screen.
Returns
Nothing.
